# App Naming Ideas

Current placeholder name: **Homelab Hub**

## Project Summary

A native iOS companion app for homelab enthusiasts that:
- Connects to self-hosted Homepage dashboards
- Displays services in a polished native grid with health monitoring (online/offline status)
- Provides an embedded browser with persistent tabs for accessing services
- Uses iOS 26's Liquid Glass design language
- Supports self-signed SSL certificates common in homelabs
- Offers custom backgrounds via photo library or Apple Intelligence AI generation
